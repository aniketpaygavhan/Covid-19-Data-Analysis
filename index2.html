COVID-19 Data Analysis Project

Introduction

This project provides an analysis of COVID-19 data to understand the trends and impacts of the pandemic. It includes data summaries and visualizations to help interpret the spread and control measures over time.

Data Summary

The data used in this analysis includes information on confirmed cases, recoveries, and deaths from various regions. Below is a brief summary of the dataset:

  • Total Entries: 18,110
  • Data Columns: 9
  • Data Source: Kaggle COVID-19 Dataset
